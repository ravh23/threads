
Feature List:

- Tech stack: MERN + Socket.io + Chakra UI
- Authentication & Authorization with JWT
- Create Post
- Delete Post
- Like/Unlike Post
- Comment to a Post
- Follow/Unfollow Users
- Freeze Your Account
- Dark/Light Mode
- Completely Responsive
- Chat App With Image Support
- Seen/Unseen Status for Messages
- Notification sounds
- Deployment for FREE



### Build the app

```shell
npm run build
```

### Start the app

```shell
npm start
```
